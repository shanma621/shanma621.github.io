Post-COVID Rent Trends Analysis

County-level rent trends before vs. after COVID, using Zillow rent data combined with Census income data to explore affordability through time-series alignment and ratio analysis.

Read the Summary View Live Report View Code

Overview

This project analyzes U.S. county-level rent trends before and after COVID-19 by combining Zillow rental time series data with Census income data, then measuring affordability using trend and ratio-based comparisons.

What I Did

  • Cleaned and validated multi-source datasets with inconsistent formats and time coverage
  • Aligned county-level time series so pre/post comparisons are actually meaningful
  • Computed rent-to-income measures and summarized changes across time
  • Built clear visuals to show trend shifts and affordability changes
  • Documented steps for reproducibility (so it’s not “trust me bro” analytics)

Tools & Skills

  • Python: pandas, NumPy
  • Data wrangling: joins, reshaping, filtering, handling missing values
  • Exploratory analysis + visual storytelling
  • Reproducible notebook workflow

Note: the detailed methodology and full outputs are available in the Live Report link (GitHub Pages).

Approach

Data Wrangling Pipeline

  • Loaded and standardized column formats across datasets
  • Validated county identifiers and joined datasets safely (no silent row loss)
  • Aligned time series to comparable periods for pre/post analysis
  • Calculated affordability metrics (rent-to-income style ratios)
  • Generated summary tables and visuals used in the report

Why This Matters

Housing affordability is a practical policy + personal-finance problem. Comparing rent in isolation can mislead; adding income context helps show whether rent increases are outpacing what people can realistically afford.

Results & Takeaways

What the Analysis Shows

  • Clear shifts in rent trends after the COVID period (direction and strength vary by location)
  • Affordability is better captured when rent is compared against income context
  • County-level patterns are not uniform—aggregates can hide big differences

What I’d Do Next (if this were work)

  • Add regional grouping (state/metro) to show patterns without losing county detail
  • Track inflation-adjusted measures to avoid “nominal dollars lie” problems
  • Build a small dashboard view for filtering counties and comparing timelines
